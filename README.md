# Twenty CRM

A modern open-source CRM.

---

## Step-1 Installation
```bash
git clone https://github.com/mhdhaidarah/TwentyCRM.git
cd TwentyCRM/
```

## Step-2 Edit the .env file to set your device IP and other settings
```bash
sudo nano .env
```
## Step-3 Just Run it Up
```bash
docker compose up -d
```
